4 Program Description
5 ----------------------------------------------
6 
7 This program is an example of steps to allow embedded database connectivity to a C++ program. The following has been checked:
8 
9 - Downloading, building and linking Poco library
10 - Downloading and linking SQLite library
11 - Using different aspects of Poco::Data wrapper according to [Poco Data User Guide](https://pocoproject.org/docs/00200-DataUserManual.html). Particularly, the following interesting features has been tested:
12  - connection, session, __binding custom objects__
13  - using Poco data types (incl. [timestamps](https://pocoproject.org/docs/Poco.Timestamp.html))
14  - some peculiarities of SQLite3: foreign key support
15

25 Building Tips
26 ----------------------------------------------
27 
28 - __POCO__ library is needed:
29  [Download complete edition](https://pocoproject.org/download/index.html)
30 - __SQLite__ library is needed:
31  [Download 64-bit DLL (x64) and command-line tools](http://www.sqlite.org/download.html)
32 - Set environmental variables:
33  `CPP_POCO_LIB_ROOT`,
34  `CPP_SQLITE_LIB_ROOT`
35 - Use your modified __assets/setenv.bat__ (run as admin) to accomplish that
36 
37 - Build POCO on x64 (on win32 it builds without problems):
38  1. in Developer Command Prompt (%comspec% /k ""C:\VS2015\Common7\Tools\VsDevCmd.bat"); cd C:\VS2015\; run _vc/vcvarsall.bat amd64_
39  2. cd poco_dir
40  3. in __component__: delete not needed components
41  4. in __build_vs140.cmd__:
42  buildwin 140 build shared both *x64* samples
43  5. run __build_vs140.cmd__
44 
45 - Set Project Properties
46  1. Configuration Properties > Debugging > Environment
47  - `PATH=%PATH%;$(CPP_POCO_LIB_ROOT)\bin64\;$(CPP_SQLITE_LIB_ROOT)\;`
48  - these are paths to dll libs PocoFoundation64, PocoData64, PocoDataSQLite64, sqlite3
49  2. Configuration Properties > General > Output and Intermediate Directories
50  - `build\$(Platform)\$(Configuration)\bin\`
51  - `build\$(Platform)\$(Configuration)\obj\`
52  3. C/C++ > General > Additional Include Directories
53  - `$(CPP_POCO_LIB_ROOT)\Foundation\include\;$(CPP_POCO_LIB_ROOT)\Data\include\;$(CPP_POCO_LIB_ROOT)\Data\SQLite\include\;`
54  4. Linker > General > Additional Library Directories
55  - `$(CPP_POCO_LIB_ROOT)\lib64\;`
